# next.js
next.js複製元

# how to install

### 1.`docker compose build`
### 2.`docker compose run frontend yarn install`
### 3.`docker compose run backend rails db:create`

### 確認方法

### rails `http://localhost:3000/`
### next.js `http://localhost:8000/`
